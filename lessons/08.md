## The network stack
![tcp stack](https://i.stack.imgur.com/L4hTM.png)

The simple stack is made up of 4 layers:
* Defines protocols and how programs interface with the transport layer services to use the network (HTTP, FTP. SMTP, etc.)
* Provides communication session management between host computers (cient/server) using TCP/UDP
* Packages data into IP datagrams, which contain source and destination IP addresses. Used forward the datagram between hosts (switching) and across networks (routing)
* Specifies details of how data is physically sent through the network (MAC address)
